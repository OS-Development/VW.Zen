

The Zen Viewer is a v3.xx viewer forked from the SL viewer-development
repository. It contains all the latest shines and fixes plus a consolidation
of features from other viewers geared towards content creation on the grid.

Features included in the Zen Viewer:

- The Original Pie Menu.
- Local Sculpt and Texture Browsing.
- Temp Sculpt and Texture Uploads.
- Fetch Inventory at Login.
- Resized View/Cam Controls Floater.
- First Run and Default Settings.
- Derender.


Also included are some nice performance enhancements:

- Kirsten's OpenJPEG Library.
- NickyD's HACD Mesh Decomposition.
